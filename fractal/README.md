Fractal benchmark
-----------------

This benchmark generates an ASCII output of a fractal. It is a fun benchmark
that I originally copied from JRuby's repository.

Added Go, Swift and Dart versions to match the Ruby one.

This benchmark also serves as little samples of those languages. That's why I
wanted to get it out here on GitHub.
